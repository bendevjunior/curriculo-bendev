01 - Programação para Ciência de Dados (2024)

Exploração profunda de Python aplicado à ciência de dados, abordando desde tipos de dados básicos até estruturas de dados avançadas como listas, tuplas, conjuntos, e dicionários. Inclui também técnicas de vetorização e o uso eficiente de bibliotecas como Pandas, Numpy e bibliotecas de visualização como Matplotlib e Plotly.
